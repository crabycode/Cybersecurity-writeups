# Cybersecurity Write-ups

A collection of cybersecurity and network security write-ups focused on protocols, attack vectors, risks, and defensive measures.

This repository contains research-based write-ups on topics such as network protocols, wireless security, common attack techniques, and practical defense strategies. Some of the materials were originally developed as university coursework and later adapted into a more structured portfolio format.

## Purpose

The goal of this repository is to demonstrate my understanding of core cybersecurity concepts, especially in areas related to networking, protocol behavior, attack surfaces, and defensive thinking.

The write-ups are focused on explaining not only how certain technologies work, but also how they can be abused and how they can be protected.

## Topics Covered

| Write-up                      | Focus                                                                         |
| ----------------------------- | ----------------------------------------------------------------------------- |
| UDP Protocol Analysis         | How UDP works, where it is used, advantages, limitations, and security risks  |
| Wi-Fi Attacks and Defenses    | Common wireless network attack techniques and practical mitigation strategies |

## Skills Demonstrated

* Understanding of networking and security fundamentals
* Ability to analyze protocols from a security perspective
* Research and technical writing
* Identifying common attack vectors
* Explaining defensive measures and risk reduction strategies
* Organizing cybersecurity knowledge in a clear and structured way

## Educational and Ethical Use

All materials in this repository are intended strictly for educational, research, and defensive cybersecurity purposes.

The goal is to understand security risks in order to prevent, detect, and mitigate them. Any offensive techniques discussed are presented from a defensive and awareness-building perspective.
